String Methoden

Strings sind Objekte. Im Kapitel über objektorientiere Programmierung werden wir sehen, wie wir Objekten bestimmte Fähigkeiten zuordnen können (Methoden). Beispiele für die eingebauten Methoden von String-Objekten:

TutorialsPoint 
>>> s = 'Habicht'
>>> s.upper()
'HABICHT'
>>> s
'Habicht'
>>> a = s.upper()
>>> a
'HABICHT'
>>> a.isupper()
True
>>> s.isupper()
False
>>> s.islower()
False


>>> s = 'maus'
>>> a = s.capitalize()
>>> a
'Maus'
>>> a.swapcase()
'mAUS'
>>>

>>> a = 'Maus'
>>> b = 3*a
>>> b
'MausMausMaus'
>>> b.index('u')
2
>>> b.index('w')

  File "", line 1, in 
    b.index('w')
ValueError: substring not found
>>> b.find('u')

>>> b.find('w')

>>> s = '  hallo  '
>>> s.strip()

>>> s = s.strip()

'hallo'
>>> s.rjust(10)
'     hallo'

(Quiz 8)

Mit der String-Methode format können wir eine formatierte Ausgabe erzeugen.

 

>>> "Es ist {}:{} Uhr".format(13, 37)
Es ist 13:37 Uhr
Wird als Formatangabe eine einfache Ganzzahl verwendet, legt sie die minimale Breite fest. Dies nutzt man für tabellarische Ausgaben. Längere Ausgaben werden nicht abgeschnitten. 
f = "{:15}|{:15}"
print(f.format("Vorname", "Nachname"))
print(f.format("Florian", "Kroll"))
print(f.format("Ramin", "Shirazi-Nejad"))
print(f.format("Sven", "Bisdorff"))
print(f.format("Kaddah", "Hotzenplotz"))
Ausgabe: 
Vorname        |Nachname
Florian        |Kroll
Ramin          |Shirazi-Nejad
Sven           |Bisdorff
Kaddah         |Hotzenplotz

Mit den Zeichen <, >, ^ lässt sich die Ausrichtung kontrollieren. Auch die Anzahl der Dezimalstellen kann kontrolliert werden:

print('Die Zahl {} und den String {} ausgeben'.format(3.1415, 'Habicht'))
print('Die Zahl {:10} und den String {:10} ausgeben'.format(3.1415, 'Habicht'))
print('Die Zahl {:<10} und den String {:>10} ausgeben'.format(3.1415, 'Habicht'))
print('Die Zahl {:10.3} und den String {:^20} ausgeben'.format(3.1415, 'Habicht'))
Ausgabe: 
Die Zahl 3.1415 und den String Habicht ausgeben
Die Zahl     3.1415 und den String Habicht    ausgeben
Die Zahl 3.1415     und den String    Habicht ausgeben
Die Zahl       3.14 und den String       Habicht        ausgeben
