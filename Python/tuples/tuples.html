Tupels

  • eine angeordnete Sequenz von Elementen, zugreifbar durch Index, unterschiedliche Datentypen sind möglich
  • immutable, d.h. Elemente können nicht verändert werden
  • repräsentiert durch runde Klammern
  • da (2) auch die normale Klammer sein könnte, wird ein 1-elementiges Tupel durch (2,) repräsentiert.

Werte tauschen: der Ausdruck x,y ist eine Kurzform von (x,y)

>>> x = 2
>>> y = 4
>>> x,y
(2, 4)
>>> x,y = y,x
>>> x,y
(4, 2)

Mit einem Tupel kann eine Funktion mehr als nur einen Wert zurückgeben.

def quotient_und_rest(x,y):
	q = x // y
	r = x % y
	return q, r


>>> quotient_und_rest(11,3)
(3, 2)
>>> type(quotient_und_rest(11,3))
<class 'tuple'>

>>> quot,rest = quotient_und_rest(11,3)
>>> quot
3
>>> rest
2
>>> rest = quotient_und_rest(11,3)[1]  # nur die zweite Variable auffangen
>>> _,rest = quotient_und_rest(11,3)   # geht auch so

>>> rest
2
