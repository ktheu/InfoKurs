Mit break verlassen wir die aktuelle Schleife. 

x = 2
while True:
    print(x)
    if x == 4: break
    x+=1

2
3
4

Mit continue wird wieder an den Beginn der Schleife gesprungen. Der Rest des aktuellen Durchgangs wird nicht mehr durchlaufen.

for i in range(8):
    if i % 2 == 0: continue
    print(i)

1
3
5
7
