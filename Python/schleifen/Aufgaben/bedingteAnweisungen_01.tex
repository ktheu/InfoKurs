\question[3] 
Was erscheint auf der Konsole, wenn das Programm nacheinander mit folgenden
Eingaben gestartet wird?
a. 12  ~~ b. 5  ~~ c. 2
\begin{lstlisting}
a = int(input())
if a > 20:
    print('A')
elif a >= 5:
    print('B')
elif a > 10:
    print('C')
else:
    print('D')
\end{lstlisting}
\begin{solutionbox}{1cm}
a. B ~~ b. B ~~ c. D
\end{solutionbox}
