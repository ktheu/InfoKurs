\question[2] Was erscheint auf der Konsole, wenn das Programm nacheinander mit folgenden
Eingaben gestartet wird?
a. 30   ~~  b. 40
\begin{lstlisting}
x = int(input())
if x > 20:
    print('A')
elif x > 10:
    print('B')
else:
    print('C')
if x > 30:
    print('D')
\end{lstlisting}
\begin{solutionbox}{1cm}
a. A   ~ b. A, D
\end{solutionbox}
