\question[4] Schreibe eine Funktion zaehl01, die zurückgibt, wieviele Zahlen
es zwischen 100 und 999 (einschließlich) gibt, die durch 3 aber nicht durch 5
teilbar sind.
\begin{solutionbox}{4cm}
\begin{lstlisting}
def zaehl01():
    zaehl = 0
    for k in range(100,1000):
        if k % 3 == 0 and k % 5 != 0:
            zaehl+=1
    return zaehl
\end{lstlisting}
\end{solutionbox}
