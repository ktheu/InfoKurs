\question[2] Schreibe eine Funktion func01, der ein String übergeben wird und
die den String ohne erstes und letztes Zeichen zurückgibt.
\begin{lstlisting}
>>> func01('Hallo')
'all'
\end{lstlisting}
\begin{solutionbox}{4cm}
\begin{lstlisting}
def func01(s):
    return s[1:-1]
\end{lstlisting}
\end{solutionbox}
