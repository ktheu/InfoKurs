\question[3] Schreibe eine Funktion zaehl02, die einen nicht leeren String erhält und zurückgibt,
wieviele Ziffern (0-9) in dem String vorkommen.
\begin{lstlisting}
>>> zaehl02('ab12cc3d60bb')
5
\end{lstlisting}
\begin{solutionbox}{4cm}
\begin{lstlisting}
def zaehl02(s):
    zaehl = 0
    for c in s:
        if c in '0123456789':
            zaehl+=1
    return zaehl
\end{lstlisting}
\end{solutionbox}
