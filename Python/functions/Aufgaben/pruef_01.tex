\question[2] Schreibe eine Funktion pruef01, der ein String mit Länge >= 2
übergeben wird und die prüft, ob die ersten beiden Zeichen gleich sind.
\begin{lstlisting}
>>> pruef01('aab')
True
>>> pruef01('abb')
False
\end{lstlisting}
\begin{solutionbox}{4cm}
\begin{lstlisting}
def pruef01(s):
    return s[0] $==$ s[1]
\end{lstlisting}
\end{solutionbox}
