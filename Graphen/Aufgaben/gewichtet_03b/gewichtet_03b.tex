\question[4]
Gib für den abgebildeten Graphen die Implementierung in Python als
ein dictionary von dictionaries an.

\includegraphics[height=4cm]{\pfad/Graphen/Aufgaben/gewichtet_03b/gewichtet_03b.png}
\begin{solutionbox}{5cm}
\begin{lstlisting}
G = {
'a': {'b':3, 'c':1},
'b': {'c':4, 'd':5},
'c': {},
'd': {'b':2},
}
\end{lstlisting}
\end{solutionbox}
