\question[3]
Gib für den abgebildeten Graphen die Implementierung in Python als
ein dictionary von sets an.

\includegraphics[height=3cm]{\pfad/Graphen/Aufgaben/ungewichtet_03a/ungewichtet_03a.png}
\begin{solutionbox}{4cm}
\begin{lstlisting}
G = {
'a': set('bc'),
'b': set('ace'),
'c': set('abd'),
'd': set('ce'),
'e': set('bd')
}
\end{lstlisting}
\end{solutionbox}
