\question[3]
Gib für den abgebildeten Graphen die Implementierung in Python als
ein dictionary von sets an.

\includegraphics[height=3cm]{\pfad/Graphen/Aufgaben/ungewichtet_03b/ungewichtet_03b.png}
\begin{solutionbox}{5cm}
\begin{lstlisting}
G = {
'a': set('bcd'),
'b': set('ac'),
'c': set('abde'),
'd': set('ac'),
'e': set('c')
}
\end{lstlisting}
\end{solutionbox}
