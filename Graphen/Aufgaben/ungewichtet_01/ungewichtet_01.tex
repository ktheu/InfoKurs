\question[3]
Gib für den abgebildeten Graphen die Implementierung in Python als
ein dictionary von sets an.

\includegraphics[height=3cm]{\pfad/Graphen/Aufgaben/ungewichtet_01/ungewichtet_01.png}
\begin{solutionbox}{5cm}
\begin{lstlisting}
G = {
'a': set('bcdef'),
'b': set('ce'),
'c': set('d'),
'd': set('e'),
'e': set('f'),
'f': set('cgh'),
'g': set('fh'),
'h': set('fg')
}
\end{lstlisting}
\end{solutionbox}
