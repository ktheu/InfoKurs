\question[3]
Gib für den abgebildeten Graphen die Implementierung in Python als
ein dictionary von sets an.

\includegraphics[height=3cm]{\pfad/Graphen/Aufgaben/ungewichtet_03/ungewichtet_03.png}
\begin{solutionbox}{5cm}
\begin{lstlisting}
G = {
'a': set('bcd'),
'b': set('ca'),
'c': set('ade'),
'd': set('ace'),
'e': set('cd')
}
\end{lstlisting}
\end{solutionbox}
