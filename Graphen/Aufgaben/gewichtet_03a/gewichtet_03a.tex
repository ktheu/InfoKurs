\question[4]
Gib für den abgebildeten Graphen die Implementierung in Python als
ein dictionary von dictionaries an.

\includegraphics[height=4cm]{\pfad/Graphen/Aufgaben/gewichtet_03a/gewichtet_03a.png}

\begin{lstlisting}
G = {
'a': {'c':4, 'd':2},
'b': {'a':3},
'c': {'a':1, 'd':5},
'd': {},
}
\end{lstlisting}
\end{solutionbox}
