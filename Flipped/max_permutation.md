[Problemstellung](https://youtu.be/Rdp_fl50GSI) - [Beispiel](https://youtu.be/CU2eECLB0J8)

    [1, 2, 0, 0, 7, 7, 5, 5]

**A2**

    [6, 8, 8, 0, 1, 1, 5, 5, 5]

**Lösungen**
